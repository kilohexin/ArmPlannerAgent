# 开发记录

## 2026-07-03

- 替换为课题组公开页面中的乐白 LM3 GLB 模型，并按 GLB 层级映射 `Joint1` 到 `Joint6`。
- 从 GLB 中提取关节轴、关节局部偏移和 `grabber` TCP 近似节点，修正末端轨迹与夹持物体跟随点不一致的问题。
- 新增 `lm3Kinematics.ts`：提供 LM3 正运动学、课程级数值 IK、笛卡尔路点到关节轨迹的转换。
- 规划 JSON 扩展为 PlanV2：保留 `steps`、`trajectory`，新增 `cartesianWaypoints`、`jointTrajectory`、`robotModel`、`executionMode`。
- 修复中文界面乱码，前端交互改为中文，Plan/Validate/Repair/Execute 保持英文流程标签。
- 执行日志增加输入 Scene Graph JSON 和规划响应 JSON，便于课程论文截图和问题定位。
- 增加手外眼虚拟相机：在 3D 场景外设置固定相机，可拍摄当前场景截图，并作为可选多模态视觉提示发送给 LLM。
- LLM 请求扩展为兼容文本/图像两种模式：无截图时发送纯文本 Scene Graph，有截图时发送 `text + image_url` 组合输入。
- 将物体释放后的状态改为静态几何放置，修复方块堆叠时被物理引擎滑落的问题。
- 验证：`npm.cmd test` 通过 7 个测试文件 20 个用例；`npm.cmd run build` 通过。
